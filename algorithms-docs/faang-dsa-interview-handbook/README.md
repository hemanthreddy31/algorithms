## Pattern Identification Quick Map
- Contiguous sums/counts -> Prefix + HashMap
- Longest constrained segment -> Sliding Window
- Next greater/smaller -> Monotonic Stack
- Weighted shortest path -> Dijkstra / Bellman-Ford / Floyd-Warshall
- Repeated overlapping states -> Dynamic Programming
